Multi-Band GPS Signal Modeling and Transmission

Project Overview

Title: Multi-Band GPS Signal Modelling and Transmission via Software Defined Radio (L1, L2, L5)

Students: Haram Najam (FA22-BCE-041) and Laiba Qureshi (FA22-BCE-033)

Supervisors: Dr. Malik Sumair Asghar and Co-Supervisor Dr. Zahid Mehmood Jehangiri

Collaboration: National Radio Telecom Corporation (NRTC)

Introduction

GPS multi-frequency signal emulation and the design of a USRP SDR-based transmitter for safe laboratory testing. Provides a platform to test receiver performance and research spoofing/jamming scenarios.

Motivation

  • Rising Dependence on GPS: Critical for aviation, military, and infrastructure; single-frequency GPS is vulnerable to jamming and spoofing.
  • Advantages of Multi-Band GPS: Utilizing L1, L2, and L5 signals offers improved accuracy, security, and robustness.
  • Need for Lab-Scale Testbed: Safe study of multi-frequency GPS and development of countermeasures using SDR and LabVIEW.

Problem Statement

  • No safe indoor setup for GPS testing; outdoor transmission is illegal.
  • Multi-band testing capabilities are not commonly available in labs.
  • Real GPS signals cannot be modified, limiting receiver performance evaluation.

Proposed Solution

Hardware/Software Requirements: USRP SDR (B210/X310), LabVIEW, NI-USRP LabVIEW Toolkit, SMA cables/attenuators.

Generates the 1023-bit C/A code used to uniquely identify each satellite. This array spreads the GPS signal for proper receiver detection.

Creates standard 1500-bit frames containing satellite position, clock corrections, and health data for realistic receiver testing.

Generates the L1 carrier wave at 1575.42 MHz and modulates the digital signal using BPSK for transmission.

Simulation Details and Results

PRN code arrays and navigation messages were successfully generated, enabling accurate multi-band GPS signal emulation.

Impact of the Project

  • Societal: Enhances navigation safety, reduces dependency on foreign tools.
  • Environmental: Lab-based testing minimizes real-world RF interference.
  • Lifelong Learning: Develops skills in SDR, signal processing, and GPS technologies.

Deliverables

  • Fully functional multi-band GPS signal generator with LabVIEW dashboard.
  • Documentation and analysis of TTFF, accuracy, and SNR measurements.
