Namaz Reminder Project Report

Project Overview

The "Namaz Reminder" is a web-based application designed to assist users in keeping track of their daily prayers. The system provides accurate prayer timings, Qaza prayer scheduling, Azkar (remembrance of Allah), and timely notifications. The application is user-friendly, interactive, and ensures that users never miss their prayers.

Objective

The main objective of this project is to create a comprehensive tool that helps Muslims stay consistent with their daily and congregational prayers. It aims to provide accurate prayer timings based on the user's location and assist in managing missed (Qaza) prayers efficiently.

Key Features

Location Detection

Automatic detection of user's city or manual input for accurate prayer times.

Prayer Timings

Displays all five daily prayers (Fajr, Dhuhr, Asr, Maghrib, Isha) and Jumma prayer with precise timings.

Notifications

Reminds users of upcoming prayers and missed prayers with daily notifications.

Azkar & Dhikr Counter

Provides daily morning, evening, after prayer, and general Azkar with a counter to track Dhikr repetitions.

Qaza Namaz Scheduler

Helps users calculate and schedule missed prayers based on available completion days.

Dynamic Scheduling

Automatically switches to the next prayer tab after setting location for smooth user experience.

Technologies Used

  • HTML5 - Structure of the web application.
  • CSS3 - Styling, layout, responsive design.
  • JavaScript - Functionality including location detection, prayer time calculation, Qaza scheduling, and notifications.
  • Font Awesome - Icons for UI enhancement.
  • Google Fonts - Poppins and Playfair Display fonts for a modern look.

Functionalities

  • Automatically detects the user's location or allows manual input for accurate prayer timings.
  • Displays prayer times and descriptions for each prayer including Fajr, Dhuhr, Asr, Maghrib, Isha, and Jumma.
  • Enables marking prayers as complete and tracks daily prayer stats.
  • Azkar section with category selection and Dhikr counter for spiritual tracking.
  • Qaza Namaz calculator for scheduling missed prayers and generating a daily plan.
  • Notifications for prayers and missed prayers, enhancing punctuality.
  • Live time display and countdown for the next prayer.

Conclusion

The Namaz Reminder project successfully integrates technology with spiritual practices, making it easier for users to manage and stay consistent with their prayers. It is a practical tool for Muslims seeking to improve punctuality and consistency in their religious obligations while providing an interactive and modern interface.
